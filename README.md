# HXRO Music Group, LLC

Welcome to the official website for HXRO Music Group, LLC, a forward-thinking record label dedicated to nurturing talent and pushing the boundaries of music.

## Features

- **About**: Learn about our mission and vision
- **Artists**: Discover the talented artists we work with
- **Releases**: Check out our latest music releases
- **Contact**: Get in touch with us

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Local Development

To run the site locally:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.

## Deployment

This site is hosted on GitHub Pages. Push changes to the `main` branch to deploy.
